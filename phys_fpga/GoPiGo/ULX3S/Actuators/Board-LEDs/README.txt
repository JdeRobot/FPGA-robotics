ICESTUDIO BLOCK: DRIVER FOR ULX3S BOARD LEDS
============================================

INPUTS:
- SIG0..SIG7: 
    8 input signals, connected to ULX3S board LEDs 1..8 
    (any of these signals can be connected to any output wire from your circuit) 

OUTPUTS:
- LED0..LED7: 
    8 output signals, to be connected to ULX3S board LEDs 1..8 
    (any of these signals can be connected to any ULX3S board LED) 

NOTES:
- An example of use is included
